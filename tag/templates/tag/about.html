MOVIE TAG

Not sure what movie to watch? Answer some questions and let me suggests a movie for you.

This project made use of movie tag data from MovieLens Tag Genome Dataset. The project was inspired by the idea of movie recommendation based on sentiments. For those who are interested, you can read more about how this app works on my blog or check out the code on the movie recommendation app on Github.
